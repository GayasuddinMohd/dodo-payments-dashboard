# SaaS Product Analytics Dashboard

## Overview
End-to-end product analytics case study for a SaaS business using SQL and an interactive Streamlit dashboard to analyze revenue, growth trends, and product performance.

## Live Dashboard
https://dodo-payments-dashboard-upcymufrsfdvc7yncepnae.streamlit.app/


## Key Features
- SQL analysis: revenue breakdown, monthly trends, top products, profitability
- Interactive dashboard: product trends, country & segment analysis
- Business insights and recommendations

## Key Insights
- A small set of products contributes most of the revenue
- Enterprise segment generates more stable revenue than SMB
- Certain regions show consistently higher growth potential

## Tech Stack
SQL, Python, Pandas, Streamlit, Plotly
